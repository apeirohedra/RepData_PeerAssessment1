A strategy for filling in all of the missing values in the dataset. The mean is used as a good strategy.

fill <- function(steps, interval) {
    filled <- NA
    if (!is.na(steps))
        filled <- c(steps)
    else
        filled <- (ave[ave$interval==interval, "steps"])
    return(filled)
}

Create a new dataset that is equal to the original dataset but with the missing data filled in.

filled <- rawdata
filled$steps <- mapply(fill, filled$steps, filled$interval)

Are there differences in activity patterns between weekdays and weekends?

Create a new factor variable in the dataset with two levels – “weekday” and “weekend” indicating whether a given date is a weekday or weekend day.

dayorand  <- function(date) {
    day <- weekdays(date)
    if (day %in% c("Monday", "Tuesday", "Wednesday", "Thursday", "Friday"))
        return("weekday")
    else if (day %in% c("Saturday", "Sunday"))
        return("weekend")
    else
        stop("invalid date")
}

Create a new factor variable

filled$date <- as.Date(filled$date)
